# Tier List

A desktop tier list app. Follow the steps below to get your TierList.exe —
no installs required on your machine.

---

## How to get TierList.exe

### Step 1 — Create a free GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Go to https://github.com/new
2. Name it **tierlist** (or anything you like)
3. Leave it **Public**
4. Click **Create repository**

### Step 3 — Upload these files
1. On the next page click **uploading an existing file**
2. Drag in everything from this zip — the `src/` folder,
   `.github/` folder, `package.json`, and this `README.md`
3. Click **Commit changes**

### Step 4 — Wait ~3 minutes
GitHub automatically builds your exe.
Watch for the orange dot → green tick at the top of your repository page.

### Step 5 — Download TierList.exe
1. Click the **Actions** tab in your repository
2. Click the latest run (should show a green ✅)
3. Scroll down to **Artifacts**
4. Click **TierList** to download a zip

Extract `TierList.exe` from that zip — done. You never need to repeat this
unless you want to change the app.

---

## Using the app

1. Put `TierList.exe` anywhere you like and double-click it
2. An `images\` folder appears next to the exe automatically
3. Drop your images into that folder
4. Click **🔄 Reload** in the app to load them

Progress is saved automatically to `tierlist_save.json` next to the exe.

---

## Controls

| Action | How |
|---|---|
| Place image | Drag the large card into a tier row |
| Skip image | Click **Next →** |
| Remove from tier | Hover over a placed image → click ✕ |
| Rename / recolour tier | Double-click the coloured tier label |
| Reorder tiers | ↑ ↓ buttons on the right of each tier |
| Add a tier | **+ Add Tier** |
| Open images folder | **📁 Open Images Folder** button |
| Export PNG | **⬇ Export PNG** |

## Supported image formats
JPG · PNG · GIF · WEBP · BMP · SVG · AVIF
